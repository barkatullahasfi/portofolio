# SKPD / Diskominfo Web Application 
 
 ## Overview 
 This project represents public-facing and internal web applications developed 
 for a local government department (SKPD) to support information services and 
 internal operational workflows. 
 
 The applications are used in a production environment and require stability, 
 security awareness, and clear accountability. 
 
 --- 
 
 ## My Role 
 I handled the technical responsibilities independently as an in-house developer. 
 
 - Backend feature development 
 - Application deployment and maintenance 
 - Handling updates and production issues 
 - Basic server configuration and monitoring 
 - Supporting both public and internal usage 
 
 --- 
 
 ## Tech Stack 
 - Laravel 
 - PHP 
 - MySQL 
 - Linux VPS 
 - Cloudflare (DNS, SSL, basic security) 
 
 --- 
 
 ## Responsibilities & Scope 
 - Development of backend logic and data handling 
 - Ensuring application stability in production 
 - Managing deployments and updates 
 - Responding to issues in a live environment
