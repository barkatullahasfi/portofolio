# ERaperda System 
 
 ## Overview 
 ERaperda is an electronic regulation management system developed for a local 
 government institution to support the drafting, management, and publication 
 of legislative documents. 
 
 The system is designed to improve transparency, document management, and 
 workflow efficiency in a regulated government environment. 
 
 --- 
 
 ## My Role 
 I worked as the primary developer with full responsibility for the system. 
 
 - Backend development using Laravel 
 - Designing and maintaining REST APIs 
 - Server deployment and configuration 
 - Basic security hardening and WAF configuration 
 - Ongoing maintenance and production support 
 
 --- 
 
 ## Tech Stack 
 - Laravel 
 - PHP 
 - MySQL 
 - Nginx 
 - Linux VPS 
 - Cloudflare (DNS, SSL, WAF) 
 
 --- 
 
 ## Key Features 
 - Authentication and role-based access control 
 - CRUD management for legislative documents 
 - Secure API endpoints 
 - Deployment with HTTPS and server hardening 
 - Protection against basic attacks using WAF and rate limiting 
 
 --- 
 
 ## Security & Access Notes 
 Public access and crawling are intentionally restricted via WAF 
 based on client compliance and data protection requirements. 
 
 This repository is provided as a **case study only** and does not contain 
 source code or sensitive configuration.
